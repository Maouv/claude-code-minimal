# Claude Code

**Claude Code** adalah CLI (Command Line Interface) resmi dari Anthropic yang memungkinkan kamu berinteraksi dengan model AI Claude langsung di terminal. Think Cursor/Copilot, tapi di terminal dan pakai Claude.

## Apa yang Bisa Dilakukan?

- Membaca, menulis, dan edit file kode secara langsung
- Menjalankan perintah terminal (git, npm, pytest, dll)
- Memahami struktur project dan navigasi codebase
- Review code, debug, dan refaktor otomatis
- Bikin PR, commit, resolve merge conflict

## Kenapa Claude Code?

- **Full context** — Claude baca semua file yang relevan, bukan cuma snippet
- **Agen otonom** — cukup kasih instruksi, dia eksekusi sendiri (baca file, edit, run command)
- **Terminal-native** — langsung di workflow dev kamu, tanpa switch app
- **Cepat** — respons real-time, streaming output, dan tool usage paralel
